Welcome to the DDA-DIA Lipid Identification Platform.

Instructions

Experiment Tab: This tab allows you to perform the lipid identification experiment.

  • Upload Files: Upload files for analysis. The database file is optional and allows you to build upon a previously existing database.
  • General Parameters: Adjust general parameter values.
  • Advanced Parameters: Adjust advanced parameter values.
  • Run Experiment: Execute the lipid identification experiment and view real-time results. You may save modified parameter selections to a file for future use.

Results Tab: Explore newly generated or previously generated results from the experiment. Select database rows and click Show Mappings to view linked features.

Usage

The DIA (Data Independent Analysis) and DDA (Data Dependent Analysis) methods complement each other to identify lipids and locate double bonds using IMS-MS data.
